## Security Hub CSPM and CIS Benchmark

### Enabled Standards
- AWS Foundational Security Best Practices
- CIS AWS Foundations Benchmark

### Purpose
Provides continuous evaluation of account configuration against security best practices.

### Architectural Significance

This introduces posture-based security monitoring in addition to:
- Log-based detections (CloudWatch metric filters)
- Behavior-based detections (GuardDuty)

This layered approach mirrors enterprise cloud security architectures.
